Projeto: Jogo de Acertar o Alvo – PIC18F4550
Este é um projeto desenvolvido durante as aulas de Microcontroladores, utilizando o kit de desenvolvimento XM118 com o microcontrolador PIC18F4550.

Descrição
O projeto consiste em um jogo simples, no qual o objetivo é acertar o alvo no momento correto. O sistema permite ao usuário:

Selecionar o volume da música de fundo;

Escolher a dificuldade do jogo (quanto maior a dificuldade, mais rápido o alvo muda de posição);

Definir o número de rodadas.

À medida que o jogo avança, a velocidade de troca da posição do alvo aumenta de acordo com a dificuldade escolhida e o número da rodada atual, tornando o desafio progressivamente mais difícil.

Tecnologias e Ferramentas
Linguagem C

MPLAB X IDE

XC8 Compiler

Microcontrolador PIC18F4550

Kit XM118
